## 部署到github与coding配置

``` bash
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo:
      github: https://github.com/.../....github.io.git,master
      coding: git@git.coding.net:.../....git,coding-pages
  branch: master
```
